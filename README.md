SmartEvent is a web-based event planning system developed as a Final Year Project (FYP2).
The system assists users in planning events through guided proposals, budgeting, venue selection,
activity recommendations, and community feedback features.

## Technology Stack
- Framework: Laravel 12
- Programming Language: PHP 8.3
- Database: MySQL
- Frontend: Blade, Bootstrap
- Server: Laragon

## System Roles
- User: Create event proposals, explore venues, budgets, activities, and interact with community features.
- Admin: Review and approve proposals, manage venues, budgets, activities, and publish events.

## Notes for Examiner
- Environment configuration files (.env) are excluded for security reasons.
- Vendor and node_modules folders are excluded as they are auto-generated.
- The repository demonstrates system structure, logic, and implementation.
